# Dataset

This folder contains the dataset used for the face recognition project.

The dataset is provided as a ZIP file. After extracting it, you will find a folder named **`yalefaces`**, which contains the dataset organised into the following directories:

- **train/** – Images used to train the face recognition model.
- **test/** – Images used to evaluate the model's performance.

Please extract the ZIP file before running the project and ensure the folder structure remains unchanged.
